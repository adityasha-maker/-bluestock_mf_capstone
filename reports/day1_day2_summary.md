# Day 1 and Day 2 Project Summary

During Day 1 of the Bluestock Mutual Fund Capstone Project, the project folder structure was created, dependencies were installed, and all ten datasets were successfully loaded using Pandas. Dataset structures were inspected using shape, data types, and sample records. AMFI codes were validated between fund_master and nav_history datasets, and a data quality assessment was performed. Live NAV data was also fetched from MFAPI and stored successfully.

During Day 2, the datasets were cleaned and validated. NAV history data was processed by converting dates, removing duplicates, forward-filling missing values, and validating positive NAV values. Investor transaction data was standardized and validated, including transaction types, dates, transaction amounts, and KYC status values. Scheme performance data was checked for valid return metrics and expense ratios. Cleaned datasets were loaded into a SQLite database using SQLAlchemy. A star schema consisting of dimension and fact tables was implemented, analytical SQL queries were created, and a data dictionary was prepared. All datasets passed validation checks and were prepared for the next phase of exploratory data analysis.
